# LucidMQ

This subdirectory CLI code to interact with lucidmq instances via a terminal.

## To Run the CLI

To run the CLI binary using cargo as normal

`cargo run {options}`

## Desired API

### Manage Consumer group

[ ] List all Consumer groups
[ ] Create a consumer group
